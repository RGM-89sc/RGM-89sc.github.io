git安装报错：Unable to locate package git

Sunday, 25 Mar 2018
Tags: Linux

我在Ubuntu 16.04上使用

sudo apt-get install git

安装git的时候出现了Unable to locate package git的错误

解决办法：
使用

sudo apt-get update

update完成后再安装git就不会报错了
